# \# ICU Monitoring System

# 

# \## Project Description

# The ICU Monitoring System is a Java-based healthcare application designed to monitor and manage patient vital signs in an Intensive Care Unit (ICU). The system records patient information such as heart rate, blood pressure, and oxygen level, and automatically detects abnormal conditions by generating alerts. This project demonstrates the implementation of Object-Oriented Programming concepts in Java and helps simulate real-time patient monitoring in hospitals.

# 

# \---

# 

# \## Features

# \- Add multiple patient records

# \- Monitor patient vital signs

# \- Display patient details

# \- Detect abnormal health conditions

# \- Generate automatic alerts for:

# &#x20; - Abnormal Heart Rate

# &#x20; - Abnormal Blood Pressure

# &#x20; - Low Oxygen Level

#
